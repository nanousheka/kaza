***********************Main Heading Uses*************************************

**The element comes with 2 props:
- mainHeadingBackgroundUrl: accepts the string of the background url (URL).
-mainHeadingText: accepts a string of any text (TEXT).

**To return this element, copy the link below and replace the uppercase words:

<MainHeading mainHeadingBackgroundUrl='URL' mainHeadingText='TEXT' />
