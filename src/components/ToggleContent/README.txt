***************************TOGGLE CONTENT USES*****************************

**The element takes 4 props:
- width: accepts a string that modify all component width (WIDTH)
-height: accepts a string that modify title container height (HEIGHT)
-title: accepts a string (TITLE)
-content: can be anything (CONTENT).

**To return this element, copy the link below and replace the uppercase words:
<ToggleContent title='TITLE' content='CONTENT' width='WIDTH' height='HEIGHT' />
