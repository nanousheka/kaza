********************************CAROUSEL USES********************************

The carousel will display images and have 2 customisable controllers:
-previous: if the image is the first, it will display the last image of the array.
-next: if the image is the last, it will display the first image of the array.

After importing carousel element, add those props:
- width: a string with pixels or percentage
- height: a string with pixels or percentage
- images: an array of strings (the url of the images you want to import).
- previousController: a string with image url.
- nextController: a string with image url.
